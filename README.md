# 🛍️ E-Commerce Website – RedStore

![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![Status](https://img.shields.io/badge/Project-Completed-green)

This project is a **fashion e-commerce website UI** built using **HTML, CSS, and JavaScript**.
It demonstrates the design and structure of an online shopping platform including product listings, product details, cart page, and user account page.

The project focuses on **frontend development, responsive layouts, and UI design for an online store**.

---

## 🚀 Live Demo

Once GitHub Pages is enabled, the website will be available at:

https://dinakaran-1.github.io/ecommerce-website/

## ✨ Features

✔ Responsive navigation menu
✔ Product category section
✔ Featured products display
✔ Product detail pages
✔ Product image gallery
✔ Shopping cart page
✔ Login and registration page
✔ Product rating system
